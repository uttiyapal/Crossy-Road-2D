# Crossy Road 2d

A simple 2D arcade game inspired by the classic Crossy Road, developed using **Godot Engine** and **GDScript**.

## Overview

This project recreates the core gameplay mechanics of Crossy Road in a 2D environment. Players must navigate through obstacles, avoid hazards, and travel as far as possible while achieving the highest score.

## Features

* 2D grid-based movement
* Obstacle avoidance mechanics
* Score tracking system
* Multiple level elements and hazards
* Built with Godot and GDScript
